## Setup deploy

1. Download ssh keys to user/.ssh folder, from cpanel https://c20010.sgvps.net:2083/cpsess4055673163/frontend/x3/telnet/keys/index.html

2. Add ssh keys as 
```sh
ssh-add -K ~/.ssh/id_rsa
```

3. Connect as
```sh
ssh -p18765 ttbagrou@c20010.sgvps.net
```
